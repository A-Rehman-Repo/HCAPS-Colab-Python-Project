# HCAPS-Colab-Python-Project
Purpose: The purpose of this chatbot is to evaluate patients' hospital experience survey data to support quality improvement decisions in healthcare. The dataset includes survey responses regarding patients' hospital experience from every state in the US.


For this project to work, we need to upload the proper CSV File to google drive in the appropriate name and directory ('/content/drive/My Drive/Colab Notebooks/HCAHPS-Hospital.csv')
